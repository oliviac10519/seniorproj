Help and How to Use

- Ensure the file is an .xlsx or .xls

- Ensure file has the dates you would like to analyze

- File must have a column with a unique identifier

- Be sure file has dates with months, days, and years

- Be sure file has columns with the EXACT spelling of:

~ "Seller Name"
~ "Selling Stage"
~ "Recurring ACV (USD)"
~ "Created Date"
~ "Date Closed"
